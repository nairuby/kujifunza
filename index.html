Welcome

Welcome to the Nairuby github group. You'll find links to some of the best ruby resources from this page and will hopefully use the space to learn/meet and work on interesting projects with other rubyists in the Nairobi area.

If you'd like to contribute to one of the projects, please join the github team by sending one of the admins an email.

Resources

Ruby

  • http://tryruby.org/ - great interactive tutorial that takes you step by step through some of the common ruby functions.
  • http://pine.fm/LearnToProgram/ - if you're new to programming, this is a great (free) online book that teaches you the basics of programming, using Ruby.
  • http://rubymonk.com/ - another interactive tutorial that offers some lessons (and an interface that let's you learn by writing programs)
  • http://www.codecademy.com/tracks/ruby - Code Academy now has a ruby track! Sign up and take the course.
